W3school

W3Schools — бесплатный образовательный веб-сайт для обучения кодированию онлайн. Первоначально выпущенный в 1998 году, он получил свое название от Всемирной паутины, но не связан с консорциумом W3. Предлагает курсы охватывающие все аспекты веб-разработки, а также публикует бесплатные шаблоны HTML.

2 урок

Что такое HTML? HTML означает язык гипертекстовой разметки. HTML — стандартный язык разметки для создания веб-страниц. HTML описывает структуру веб-страницы. HTML состоит из ряда элементов HTML-элементы сообщают браузеру, как отображать контент. Элементы HTML обозначают фрагменты контента, например «это заголовок», «это абзац», «это ссылка» и т. д.

3 урок

HTML (язык гипертекстовой разметки) — это код, который используется для структурирования веб-страницы и ее содержимого. Например, контент может быть структурирован в виде набора абзацев, списка маркированных пунктов или с использованием изображений и таблиц данных. Как следует из названия, эта статья даст вам базовое понимание HTML и его функций.

4 урок

HTML-атрибуты Все элементы HTML могут иметь атрибуты. Атрибуты предоставляют дополнительную информацию об элементах. Атрибуты всегда указываются в начальном теге. Атрибуты обычно представлены парами имя/значение, например: name="value"
